# Osnovy_app
App to collect osnovy data (in fin, they will be integrated to HanPolnet)

Popis výsledného datasetu: 

Proměnné: 
"id.osn" - tohle je v současné podobě dosti blbě udělané, ale bylo by ideální mít unikátní identifikátor spojující rok, beh a identitu osnov, tedy nasledující proměnné  

"rok"  - rok sběru dat, ideální by bylo mít celé datum

"beh" - 1 nebo 2 (vyjíměčně jiné číslo)

"osnova" - osnova 

"posice" - mohlo by se jmenovat bunka, od A1 do H8

"druhK" - druh rostliny, máme seznam známých druhů, ale je potřeba mít možnost přidat druh, pokud se objeví nový druh

"pocet"- počet, pro většinu rostlin to znamená počet lodyh (číslo), pro jiné to znamené presence/absence. V době kdy jsi tam byl, tak se sice počítalo všechno, ale v dalším roce to dělat nebudeme pro vysokou časovou náročnost.
